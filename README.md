# gtoc

Gtoc generates TOC(Table-Of-Content) from markdown, for README.md or GitHub's wiki page.

## What's TOC?

It is **table of content**, that would be something like:

```mkdn
* [gtoc](#gtoc)
      * [What's TOC?](#whats-toc)
      * [Usage](#usage)
      * [License](#license)
```

## Usage

```sh
# If README.md exists:
$ gtoc

# You can specify the path:
$ gtoc example/foo.md
```
